## 需要处理才能上架（30 个）

### 中国大陆（1 个）

需要提供 **ICP 备案号**才能在中国大陆 App Store 上架。


| 国家/地区 | 需要操作                            |
| ----- | ------------------------------- |
| 中国大陆  | 申请 ICP 备案号并填入 App Store Connect |


**办理方式：**

- 个人备案：不允许有社交属性，通常会被拒
- 企业备案（推荐）：注册个体工商户 → 申请 ICP 备案 → 拿到备案号填入 App Store Connect
- 流程约需 1~2 个月，费用数百元

---

**办理方式：**
