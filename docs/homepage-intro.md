Orbit 让记忆和协作都更像“和熟人分享一本私密手帐”：记录要随手、重温要轻松、协作要可控。我们把照片、语音、位置、账本和小游戏放在同一条主线里，用轻量的筛选与分享，让你能在安全、低干扰的空间里跟亲密的人对齐生活。

# 我的主页功能说明

我的主页是 Orbit 的核心落点，聚合“记录、浏览、协作”主流程：
- 头像区：左上角头像可全屏预览，再点关闭；右下角变换符号可切换随机男/女生头像，或改为自己上传。右侧展示姓名、邮箱和邀请码分享入口，下方汇总记忆总数与账本好友数量。
- 我的密友：默认展示 5 位，可展开、搜索。添加方式：① 先创建虚拟人，未来对方用邀请码注册后自动绑定；② 邀请真人，对方同意后建立关系；③ 删除后会转为虚拟好友占位。列表支持 @、查找与展开/收起的轻量交互。
- 入口与状态：随机回忆一键抽取往日记忆。设置涵盖网络策略（如仅 Wi‑Fi 上传/刷新）、隐私与屏蔽、通知、语言/区域、账号安全与登出、数据备份与导出、版本与缓存管理。Service Worker 会提示“有新版本可刷新”，点击即重载。关键状态（用户、好友、记忆、账单）由全局状态维护，切换页面或重开可快速恢复，未完成的图片上传会自动续传。底部主导航可在记忆流、地图、账本、小游戏、个人页间快速切换。

# 记忆模块

从上到下的可点击逻辑：

- 顶部提醒条：若离线、仅 Wi‑Fi 上传/刷新或有新版本，会在安全区域内提示；点击“刷新/重载”立刻应用；离线提示可跳转网络设置或关闭。（UI 提示条待接入）
- 好友筛选条：支持多选好友标签，AND 逻辑筛选记忆；“清除”恢复全量。
- 记忆发布入口：点击新建记忆；如不满足“仅 Wi‑Fi 上传”，可确认继续或延后排队。
- 视图切换：可按日期/地点/城市查看与分组。
- 回忆相册：可按日期/好友筛选，左右滑动浏览，支持生成分享海报。
- 记忆流卡片：
	- 媒体：点图片/视频可全屏预览并左右滑；语音点击播放，支持暂停/恢复。
	- 文本/元数据：时间、地点、天气、心情可点开查看详情或跳至地图；天气/心情可多选；@ 好友双方同步显示。
	- 操作：右上菜单含删除、编辑、账本绑定、复制链接/文案；删除需二次确认，编辑回到原编辑页；账本绑定后账本页同步展示。
	- 列表：上拉加载更早记忆。
	- 评论：作者和被 @ 的好友可评论/回复（文字/语音）；作者可删任何评论，被 @ 的人只能删自己的。
- 上传与录制：
	- 图片/视频上传前检查网络与体积，超限时提示压缩或分批；图片默认压缩，上传有超时防挂死策略。
	- 语音录制 30 秒上限，超时自动停止并提示可重录；上传成功后即时插入流。
- 占位与重试：上传失败的卡片会显示重试按钮与错误原因，点击重试会再次校验网络；长按可删除失败任务。

# 地图页面

- 好友筛选：支持多选好友头像，AND 逻辑过滤光点与列表；点击“全部”清空筛选，足迹数量实时更新。
- 分组视图：可在“按地点/按城市”切换，城市模式自动聚合并取均值坐标，地点模式逐点展示。
- 光点与记忆：点击光点或城市聚合卡片可展开记忆列表；列表按筛选好友交集展示 @ 或发布者为所选好友的记忆。
- 地图交互：保留缩放/拖动，顶部导航和筛选条不遮挡地图点击；加载异常时降级为占位，不阻塞页面。

# 游戏页面

- 入口：从底部导航进入小游戏聚合页，保持返回后记忆流状态不丢失。
- 内容：轻量化互动玩法，随时可退出返回主功能；运行中遵守系统静音与网络策略。
- 成绩：支持本地记录基础成绩或回合数，未联网时不阻塞退出。
- iOS 需要把铃声打开才有游戏声音。

# 账本 / 记账

- 创建方式：发布记忆时勾选“顺便记账”或在记忆卡片中绑定账本，支持个人和平摊两种模式。
- 分摊规则：平摊自动按参与人等分；个人模式仅记当前用户，日后可编辑参与人及金额。
- 展示：账本标记会出现在记忆卡片上，可从账本页查看列表、合计与参与人分摊详情。
- 同步：编辑或删除账本会同步更新记忆与账本列表；无网时会提示重试，不会悄悄失败。
- 只有自己可以知道账单，即使记忆中艾特了好友，好友也不会知道你的账单。
