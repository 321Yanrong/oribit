# 我的主页功能说明

我的主页是 Orbit 进入应用后的核心落点，聚合“记录、浏览、协作”的主流程。左上角头像点击即可全屏预览，再点一次关闭；头像右下角的变换符号可选择随机男生或女生头像，也可改为自己上传。头像右侧展示姓名和邮箱，再旁边是分享邀请码的图标，方便随时分发。头像区域下方有一行汇总当前记忆总数与账本好友数量，帮助快速掌握进度与协作范围。

“我的密友”最多先展示 5 位，可展开查看全部并支持搜索。添加好友有三种方式：1) 先创建虚拟人，给 TA 一个名字即可，未来对方注册后输入你的邀请码会直接绑定，无需重新邀请；2) 主动用邀请码邀请真人，对方同意后才会建立关系；3) 如果一方删除了对方，对面列表会把该好友标记成虚拟好友。列表对 @、查找、展开/收起都做了轻量交互，保持页面简洁。

随机回忆入口提供一键抽取往日记忆，帮助回溯灵感。设置入口覆盖网络策略（如仅 Wi‑Fi 上传/刷新）、隐私与屏蔽、通知开关、语言/区域、账号安全与登出、数据备份与导出、版本与缓存管理等。更新后的 Service Worker 会提示“有新版本可刷新”，点击立即生效并自动重载，保证主页始终使用最新资源。所有关键状态（用户、好友、记忆、账单）由全局状态管理维护，页面切换或重新打开时能秒级恢复，上次未完成的图片上传会自动尝试继续，减少重复操作。
底部保留主导航，可在记忆流、地图、账单、小游戏和个人页间快速切换。

# 记忆模块

从上到下的可点击逻辑：

- 顶部提醒条：若离线、仅 Wi‑Fi 上传/刷新或有新版本，会在安全区域内出现提示，点击“刷新/重载”即刻应用；离线提示点击可跳转网络设置或关闭提示。（待接入 UI 提示条）
- 好友筛选条：展示多选好友标签，点击标签将筛选出同时包含这些好友的记忆（AND 逻辑）；点击“清除”恢复全量。
- 记忆发布入口：点击可新建记忆，进入编辑页；若当前网络不满足“仅 Wi‑Fi 上传”，会弹出确认提示或延后队列。
- 视图切换：支持按日期、地点、城市查看与分组；相册视图可按时间/好友筛选回顾照片。

- 记忆流卡片：
	- 媒体：点图片/视频可全屏预览并左右滑；语音点击播放，支持暂停/恢复。
	- 文本/元数据：时间、地点、天气、心情可点开查看详情或跳至地图；天气/心情支持多选；@ 好友会在双方视角同步显示。
	- 操作：卡片右上可展开菜单，含删除、编辑、账本绑定、复制链接/文案等；删除二次确认，编辑回到同一编辑页；账本绑定后会在账本模块同步出现。
	- 交互状态：上拉可继续加载更早的记忆；
	- 评论：作者和被 @ 的好友可评论/回复，支持文字与语音；作者可删除任何评论，被 @ 的人只能删自己评论。
- 上传与录制：
	- 图片/视频上传前检查网络与体积，超限时提示压缩或分批；图片默认压缩，上传有超时防挂死策略。
	- 语音录制带 30 秒上限，超时自动停止并提示可重新录制；上传成功后即时插入流中。
- 占位与重试：上传失败的卡片会显示重试按钮与错误原因，点击重试会再次校验网络；长按可删除失败任务。


# 地图页面

- 好友筛选：支持多选好友头像，AND 逻辑过滤光点与列表；点击“全部”清空筛选，足迹数量实时更新。
- 分组视图：可在“按地点/按城市”切换，城市模式自动聚合并取均值坐标，地点模式逐点展示。
- 光点与记忆：点击光点或城市聚合卡片可展开记忆列表；列表按筛选好友交集展示 @ 或发布者为所选好友的记忆。
- 地图交互：保留缩放/拖动，顶部导航和筛选条不遮挡地图点击；加载异常时降级为占位，不阻塞页面。
