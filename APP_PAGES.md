# Orbit App 页面文档

> 本文档描述 Orbit App 所有页面的 UI 元素和交互功能，供开发参考。

---

## 应用整体结构

### 导航方式

应用使用**底部导航栏**在主页面之间切换。地图页（`map`）始终保持挂载状态，切换到其他页面时被隐藏但不销毁。其他三个页面以覆盖层形式渲染在地图页上方。

### 底部导航栏（BottomNav）

| 位置 | 图标 | 标签 | 说明 |
|------|------|------|------|
| 1 | FiMap | **地图** | 打开地图页 |
| 2 | FiImage | **记忆** | 打开回忆流 |
| 中间 | 相机（黄色） | （无标签） | **发布记忆**：切换到记忆页并立即打开发布编辑器 |
| 4 | FiCreditCard | **账单** | 打开账单页 |
| 5 | FiUser | **我的** | 打开个人主页 |

**特殊交互：**
- 当前激活页的图标会放大，标签加粗，下方出现小圆点
- **长按**（约 900ms）任意非发布按钮：弹出确认框「强制重启 Orbit 以清理状态？」，确认后强制重载应用

## 一、地图页（MapPage）

**文件：** `src/pages/MapPage.tsx`

**功能定位：** 以全屏地图展示所有好友的回忆足迹，每条回忆以图钉形式标注在地图上。

---

### 页面顶部

| 元素 | 说明 |
|------|------|
| 标题 | **「友情地图」** |
| 切换按钮 | **「按城市」** / **「按地点」**，切换图钉的分组方式 |
| 足迹计数 | **「N 个足迹」**（相机图标 + 数量） |

### 好友筛选栏（横向滚动）

- **「全部」** 芯片：清除筛选，显示所有好友的回忆
- 每位好友的头像 + 昵称：点击只显示该好友的图钉；支持多选

---

### 弹出层一：记忆列表（点击地图图钉后）

点击地图上的图钉后，从底部弹出该地点/城市的回忆列表。

| 元素 | 说明 |
|------|------|
| 标题 | 地点名称 |
| 副标题 | 详细地址 或「N 条记忆」 |
| 关闭按钮 | **×**（右上角） |
| 空状态 | **「暂无记忆」** |
| 回忆卡片 | 日期、**「含账单」**徽章（有账单时显示）、文字内容（无内容时显示「（无文字记录）」）、@好友标签 |

### 弹出层二：单条回忆详情（点击回忆卡片后）

| 元素 | 说明 |
|------|------|
| **「返回列表」** | 关闭详情，回到记忆列表 |
| **×** | 关闭详情和列表，回到地图 |
| 照片轮播 | **「上一张」** / **「下一张」** 箭头，底部圆点（点击可跳转），支持触摸滑动 |
| 地点 + 日期 | 发布位置和时间 |
| 标签芯片 | **「天气：…」**、**「心情：…」**、**「路线：…」** |
| @好友 | 被标记的好友列表（已删好友显示「已不是好友」） |
| 账单提示 | **「此记忆附有账单」**（有账单时显示） |

**互动区（互动摘要行）：**

- 点赞数（心形图标）+ 评论数 + 点赞者头像缩略图 + 展开箭头（`>`）
- 点击后展开互动详情：

| 互动详情元素 | 说明 |
|------------|------|
| 点赞区 | **「N 人点赞」**；**「点赞」** / **「已赞」** 按钮；无人点赞时显示「快来点亮爱心吧～」 |
| 评论区 | **「N 条评论」**；评论列表（文字 或 语音播放条）；无评论时显示「还没有评论，来发表第一条评论吧！」 |
| 评论输入 | 文本框（placeholder：「写下你的评论」）+ 发送按钮（纸飞机图标）；按 Enter 发送 |

---

### Toast 提示（自动消失）

以下情况会在页面顶部出现提示条：
- 地图服务未配置 / 当前浏览器不支持地图 / 地图加载失败
- 暂无回忆足迹
- 评论已发送 / 评论发送失败

## 二、回忆流（MemoryStreamPage）

**文件：** `src/pages/MemoryStreamPage.tsx`

**功能定位：** 时间线式的回忆信息流，展示自己和好友的所有回忆，支持发布、编辑、删除、搜索、筛选。

---

### 页面顶部

| 元素 | 说明 |
|------|------|
| 标题 | **「回忆流」** |
| 数量提示 | **「共 N 条记忆」**（有筛选时显示「找到 a / b 条」） |
| 新评论提示 | **「N 条新评论」**（开启通知时显示，点击打开通知面板） |
| **「回忆相册」** | 打开共享相册故事模式 |
| **「记录此刻」** | 打开发布编辑器（同底部中间按钮） |

### 搜索与筛选栏

| 元素 | 说明 |
|------|------|
| 搜索框 | placeholder：**「搜索内容、地点...」**，有内容时右侧出现清除（×）按钮 |
| 分组切换 | **「按日期」** / **「按城市」**（日历图标 / 城市图标） |
| 月份选择 | 显示当前月份（格式 YYYY / MM），无筛选时显示「全部」，已选时右侧出现**「清空」** |
| 排序 | **「时间 ↓」** / **「时间 ↑」** |

### 好友筛选栏（横向滚动）

- **「全部」** 芯片
- 每位好友的头像芯片，已选中的芯片显示 **「✓」**，支持多选

### 加载与空状态

- 加载中：**「加载回忆中...」** + **「如果卡住，点此重新加载」**
- 无数据：**「还没有回忆」** + **「点击「记录此刻」开始你的第一个记忆」**

---

### 回忆卡片（信息流主体）

每张卡片包含：

| 元素 | 说明 |
|------|------|
| 作者头像 + 昵称 | 发布人信息 |
| 发布时间 | 格式化时间（如「3小时前」） |
| 文字内容 | 回忆正文，含 @好友 标签 |
| 照片网格 | 最多显示 4 张，超出显示「+N」 |
| 视频提示 | **「▶ N 个视频」** |
| 语音条 | 可播放的语音评论行 |
| 记账入口 | **「记账」**（有账单时显示） |
| 互动摘要 | 点赞数 / 评论数 / 点赞者头像 + 展开箭头 |

**自己发布的回忆：**
- 编辑图标：打开编辑器
- 删除图标：弹出确认

**他人发布的回忆：**
- **⋯** 更多菜单 → **「举报发布者」**（打开举报页）/ **「屏蔽发布者」**（弹出确认）

---

### 弹出层与模态框

#### 发布 / 编辑编辑器

| 元素 | 说明 |
|------|------|
| 顶部按钮 | **「取消」**（左）/ **「保存修改」** 或 **「发布」**（右，加载时显示转圈） |
| 标题 | **「编辑回忆」** 或 **「记录此刻」** |
| 文字输入 | 回忆正文 |
| 天气 | **「那天天气」** 下拉选择 |
| 心情 | **「那天心情」** 下拉选择 |
| 路线 | **「行程路线」** + 提示「用 → 分隔地点」，placeholder：「例：机场 → 酒店 → …」 |
| 照片上传 | 图片选择器 |
| @好友 | 标记好友 |

#### 互动详情底部弹出（评论 / 点赞详情）

| 元素 | 说明 |
|------|------|
| 标题 | **「互动详情」** |
| 点赞区 | **「N 人赞过」**；头像列表 |
| 评论区 | **「N 条评论」**；每条评论可「回复」；自己的评论可「删除」 |
| 空评论 | **「暂无评论，快来抢沙发～」** |
| 回复输入 | **「说点什么...」** / **「回复 @…」**；**「发送」** 按钮；**「取消」** 取消回复 |
| 语音评论 | VoiceRecorder 组件 |

#### 时间范围筛选弹窗

| 元素 | 说明 |
|------|------|
| 标题 | **「按时间范围筛选」** |
| 副标题 | **「仅筛选回忆流，不会打开相册播放」** |
| 月份选择 | 月份列表 |
| 底部按钮 | **「清空」** / **「应用」** / **「关闭」** |

#### 新评论通知面板

| 元素 | 说明 |
|------|------|
| 标题 | **「新评论」** |
| 空状态 | **「暂无未读评论」** |
| 评论条目 | 普通评论文字 或「🎤 [语音评论]」 |

#### 举报页（ReportPage）

全屏举报表单，标题**「举报投诉」**，举报类型包括：
- 发布违规内容
- 恶意骚扰
- 冒充他人
- 垃圾广告
- 虚假信息

支持：详细描述文本 + 上传截图证据；**「提交」** 按钮。

## 三、账单页（LedgerPage）

**文件：** `src/pages/LedgerPage.tsx`

**功能定位：** 记录和统计与回忆关联的消费，支持多人 AA 分摊。

---

### 页面顶部

| 元素 | 说明 |
|------|------|
| 标题 | **「财务足迹」** |
| 月份筛选 | 当前月（格式 YYYY / MM）或「全部」，已选时显示**「清空」** |
| 排序 | **「时间 ↓」** / **「时间 ↑」** |
| **「记一笔」** | 渐变按钮，打开账单录入表单 |

### 统计摘要

- **「本月总支出」** 或 **「累计总支出」** + 金额（¥）

### 视图切换标签

- **「按记忆」**：以关联回忆为单位聚合账单，显示「N 笔记录」和「我花费 ¥…」
- **「按城市」**：以城市为单位聚合，显示「包含 N 笔消费」和「此地我共花费 ¥…」

### 空状态

**「空空如也，记下第一笔开销吧」**

---

### 弹出层：账单录入 / 编辑（LedgerModal）

| 元素 | 说明 |
|------|------|
| 顶部按钮 | **「取消」**（左）/ **「修改账单」** 或 **「记一笔」**（标题）/ **「保存」**（右，加载时转圈） |
| 消费类型切换 | **「👫 多人 AA」** / **「🙋 个人消费」** |
| 关联回忆 | 下拉选择，placeholder：**「选择要关联的记忆...」** |
| 消费条目 | 每行：消费类目芯片（饮食 / 交通 / 住宿 / 娱乐 / 购物 / 医疗 / 其他等）+ **「备注（选填）」** + 金额（点击打开计算器键盘）|
| **「添加项目」** | 新增一行消费条目 |
| 合计 | 所有条目金额之和 |
| 参与者（多人 AA 时） | **「选择参与者（默认含自己）」**；底部显示「总计 N 人均摊，每人 ¥…」 |

**计算器键盘（CalcPad）：**
- 数字键 0–9，运算符 + / - / × / ÷
- **「C」**：清除；**「←」**：退格；**「= 确认」**：确认金额

---

### 弹出层：账单详情（GroupDetailModal）

| 元素 | 说明 |
|------|------|
| 标题 | 城市名或回忆标题（无则「未命名分类」） |
| 副标题 | **「在这段行程/分类里我的开销」** |
| 消费类型徽章 | **「🙋 个人消费」** 或 **「👫 多人均摊」** + 均摊人数说明 |
| 条目列表 | 类目 + 备注 + 金额 |
| 合计 | **「总消费」** / **「我分摊的费用」** + **「共 N 人均摊」** |
| **「收起明细」** | 折叠详情 |
| 编辑 / 删除 | 底部操作按钮 |

**确认和错误提示：**
- 删除确认：**「确定删除这笔账单？」**
- 错误 alert：**「删除失败」** / **「保存失败」**

## 四、我的（ProfilePage）

**文件：** `src/pages/ProfilePage.tsx`

**功能定位：** 个人资料管理、好友系统、数据统计、应用设置入口。

---

### 主页面

| 元素 | 说明 |
|------|------|
| 左上角汉堡菜单 | 打开**设置中心**侧边栏 |
| 头像 | 点击打开头像选择弹窗 |
| 骰子图标 | 随机生成新头像 |
| 昵称 | 可直接点击编辑 |
| 邮箱 | 显示绑定邮箱（未绑定显示「未绑定」） |
| 存储空间 | 进度条，显示「已用 MB / 1024 MB」 |
| 个人签名 | **「记录生活碎片 ✨」** + 最近位置 + 最近活跃时间 |
| 数据统计行 | **「帖子」**、**「好友」**、**「账单」** 各自的数量（可点击） |

### 社交功能区

| 元素 | 说明 |
|------|------|
| **「复制邀请码」** | 复制邀请码到剪贴板 |
| 添加好友图标 | 打开添加好友弹窗 |
| **好友申请列表** | 有待处理申请 + 开启通知时显示；每条显示「想加你为好友」+ **「接受」** / **「拒绝」** |

### 我的足迹

| 元素 | 说明 |
|------|------|
| 年度统计 | **「{年} 年的记忆」** + **「共 N 天」** |
| 最常陪伴 | 最常一起出现在回忆里的好友 + 次数 |
| **「随机回忆」** | 随机展示一条历史回忆 |
| 回忆录网格 | 所有回忆的照片缩略图网格，点击可查看详情 |

### 账号操作

| 按钮 | 说明 |
|------|------|
| **「退出登录」** | 退出当前账号 |
| **「删除账号」** | 永久删除账号（二次确认） |

---

### 设置中心（汉堡菜单打开）

包含以下功能入口：

| 设置项 | 说明 |
|--------|------|
| 关于 Orbit | 版本信息、Orbit 社区公约 |
| 字体大小 | 调整全局字体 |
| 深色模式 | 切换深色 / 浅色主题 |
| 网络诊断 | 检测当前网络连接状态 |
| 帮助与客服 | 常见问题 |
| 意见反馈 | 提交反馈 |
| 绑定真实账号 | 将马甲账号绑定到真实账号 |
| 我的邀请码 | 查看和分享邀请码 |
| 更换邮箱 | 输入新邮箱 + 当前密码，发送确认邮件 |
| 修改密码 | 输入当前密码 + 新密码（至少 6 位）+ 确认新密码 |

---

### 头像选择弹窗

| 选项 | 说明 |
|------|------|
| **「男款」** | DiceBear 男性风格随机头像 |
| **「女款」** | DiceBear 女性风格随机头像 |
| **「上传头像」** | 从相册选取图片上传 |
| **「取消」** | 关闭弹窗 |

## 五、小游戏页（GamesPage）

**文件：** `src/pages/GamesPage.tsx`

> **注意：此页面目前未接入底部导航，用户无法通过正常操作访问。**

**功能定位：** 小游戏中心，包含多个内嵌小游戏。

| 元素 | 说明 |
|------|------|
| 游戏列表 | 展示可玩的小游戏入口 |
| 音效控制 | 音效开关 + 音量滑块 |
| 游戏全屏 | 点击游戏后从底部弹出全屏 Sheet，内嵌游戏组件 |

如需将小游戏接入底部导航，需要在 `src/types/index.ts` 的 `PageType` 中新增 `'games'`，并在 `src/App.tsx` 的路由逻辑和 `src/components/BottomNav.tsx` 中补充对应入口。
